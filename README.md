## 벤치마킹 프로젝트 계획의도
> 
> 시각적인 요소 확대 : 구호개발 웹사이트의 근본적인 목적인 후원에 조금 더 효과적으로 다가갈 수 있도록 시각적인 요소를 자극할 수 있는 이미지와 슬라이드를 배치하였습니다.
